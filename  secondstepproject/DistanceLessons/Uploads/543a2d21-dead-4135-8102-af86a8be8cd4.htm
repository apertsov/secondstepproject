Лабораторна робота №5 (15 балів, 2 пари)

Побудувати задану в окремому завдані  динамічну структуру даних. Забезпечити можливість вводу інформації та виводу структури на кожному етапі своєї роботи. По закінченню роботи програми вся зайнята динамічна пам’ять має бути коректно звільнена.

 

1.          Для бінарного дерева пошуку, ключами і значеннями вузлів якого є цілі числа, створити функцію яка підраховує суму всіх вузлів які знаходяться на парних рівнях.

2.          Для бінарного дерева пошуку, ключами і значеннями вузлів якого є цілі числа, створити функцію яка друкує всі значення дерева обходячи його по рівнях, причому по непарних рівнях рух відбувається зліва направо, а по парних навпаки.

3.          Для бінарного дерева пошуку, ключами і значеннями вузлів якого є цілі числа, створити функцію яка знаходить довжину найдовшої гілки дерева.

4.          Для бінарного дерева пошуку, ключами і значеннями вузлів якого є цілі числа, створити функцію яка друкує всі листові елементи дерева.

5.          Для бінарного дерева пошуку, ключами і значеннями вузлів якого є цілі числа, створити функцію яка зливає два дерева в одне.

6.          Для бінарного дерева пошуку, ключами і значеннями вузлів якого є цілі числа, створити функцію яка дає відповідь питання чи дерево є збалансованим.

7.          Для бінарного дерева пошуку, ключами і значеннями вузлів якого є цілі числа, створити функцію яка друкє елементи найдовшої гілки деорева починаючи від листового елемента і закінчуючи коренем дерева, якщо таких гілок декілька, то надрукувати будь-яку з них.

8.          Для бінарного дерева пошуку, ключами і значеннями вузлів якого є цілі числа, створити функцію яка друкє елементи найдовшої гілки деорева починаючи від кореня дерева і закінчуючи листовим елементом, якщо таких гілок декілька, то надрукувати будь-яку з них.

9.          Розв’язати попередню задачу з варіанту №1 використавши червоно-чорне дерево замість бінарного дерева пошуку.

10.       Розв’язати задачу з варіанту №2 використавши червоно-чорне дерево замість бінарного дерева пошуку.

11.       Розв’язати задачу з варіанту №3 використавши червоно-чорне дерево замість бінарного дерева пошуку.

12.       Розв’язати задачу з варіанту №4 використавши червоно-чорне дерево замість бінарного дерева пошуку.

13.       Розв’язати задачу з варіанту №5 використавши червоно-чорне дерево замість бінарного дерева пошуку.

14.       Розв’язати задачу з варіанту №7 використавши червоно-чорне дерево замість бінарного дерева пошуку.

15.       Для піраміди, ключами і значеннями вузлів якої є цілі числа, створити функцію яка для заданої піраміди будує піраміду оберненої впорядкованості.

16.       Для піраміди, ключами і значеннями вузлів якої є цілі числа, створити функцію яка будує піраміду заповнюючи її значеннями з двох масивів K (ключі) і V (значення). При цьому час роботи функції не повинен перевищувати O(n log(n)).

17.       Для піраміди, ключами і значеннями вузлів якої є цілі числа, створити функцію яка будує два масиви записуючи в них всі ключі і значення з піраміди. Перший масив - впорядкований масив ключів, другий – масив значень, причому якщо в першому масиві на i-му місці знаходиться певний ключ, то в другому масиві на цьому ж місці знаходиться відповідне йому значення. При цьому час роботи функції не повинен перевищувати O(n log(n)).

18.       Для піраміди, ключами і значеннями вузлів якої є цілі числа, створити функцію яка видаляє з піраміди довільний елемент (елемент на який вказує ітератор передається як параметр).

19.       Для піраміди, ключами і значеннями вузлів якої є цілі числа, створити функцію яка видаляє з піраміди всі вузли які містять повторні входження ключів.

20.       Для піраміди, ключами і значеннями вузлів якої є цілі числа, створити функцію яка друкує елементи піраміди здійснюючи її обхід в глибину починаючи зліва.

21.       Розв’язати задачу з варіанту №8 використавши червоно-чорне дерево замість бінарного дерева пошуку.

22.      Для піраміди, ключами і значеннями вузлів якої є цілі числа, створити функцію яка будує впорядкований масив. При цьому час роботи функції не повинен перевищувати O(n log(n)).

23.     Для бінарного дерева пошуку, ключами і значеннями вузлів якого є цілі числа, створити функцію яка перетворює дерево таким чином що вказана вершина стає коренем цього дерева (елемент на який вказує ітератор передається як параметр).

24.    Для піраміди, ключами і значеннями вузлів якої є цілі числа, створити функцію яка будує для заданої піраміди червоно-чорне дерево . При цьому час роботи функції не повинен перевищувати O(n log(n)).

25.    Для червоно-чорного дерева , ключами і значеннями вузлів якого є цілі числа, створити функцію яка вилучає з дерева вказаний елемент (елемент на який вказує ітератор, що передається як параметр) і всі його дочірні елементи. При цьому час роботи функції не повинен перевищувати O(m log(n)) , де m - кількість елементів які треба видалити.
